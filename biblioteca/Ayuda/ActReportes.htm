ACTUALIZACIÓN DE LOS REPORTES

 

Nos permite gestionar los Reportes de nuestra base de datos Biblioteca. 

Cuenta con opciones de consulta sobre cada tipo de opciones de registro, como son: Tipo de Usuario, Usuario, Carné, Libro, Tema, Empleado, Devolución y Ficha de Préstamo.

Asimismo con las siguientes opciones:           

·        Seleccionar Reporte: Permite seleccionar el registro que se desea visualizar.

·       Ejecutar Reporte: Permite visualizar el reporte.

·        Seleccionar Tipo de Impresora: Nos permite abrir el cuadro de diálogo de las propiedades de impresión para posteriormente imprimir el reporte elegido.

·       Salir: Cierra la ventana Reporte
